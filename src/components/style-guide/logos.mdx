## TintedImage Component

The tinted logos use the `TintedImage` component which applies:

- `filter: grayscale(100%) brightness(1.2) contrast(1.1)` - Converts to monochrome with adjusted brightness/contrast
- `mixBlendMode: 'screen'` on the image - Lightens the result
- A colored overlay with `background: var(--accent-primary)` and `mixBlendMode: 'multiply'` - Applies the tint
- `transform: translate3d(0,0,0)` - Forces GPU acceleration for Safari compatibility

This preserves the tonal values (light and dark areas) while applying theme colors with excellent cross-browser compatibility.

### Usage

```tsx
// Basic usage (default primary color tint)
<TintedImage src="/logos/ING_Group_N.V._Logo.svg" alt="ING" />

// Black logo (needs invert)
<TintedImage
  src="/logos/nike-vector-logo.svg"
  alt="Nike"
  filters={{ invert: true }}
/>

// Custom filters (ING logo needs special settings)
<TintedImage
  src="/logos/ING_Group_N.V._Logo.svg"
  alt="ING"
  filters={{ brightness: 1.4, contrast: 0.3 }}
/>

// Custom tint color
<TintedImage
  src="/logos/Mediamonks.svg"
  alt="MediaMonks"
  tintColor="var(--accent-secondary)"
/>
```

For more examples, see the [TintedImage Storybook stories](?path=/story/common-tintedimage--default).
