# Spacing

Our spacing system provides a consistent scale for layout, padding, margins, and gaps. All values use rem units (relative to root font size) for better scaling and accessibility.

## Spacing Scale

The scale follows a logical progression from 2px to 384px, providing flexibility for micro-adjustments to large layout spacing.

| `--spacing-1-5` | 0.375rem | 6px | Tight spacing |

## Usage Examples

### Component Padding
```css
.card {
  padding: var(--spacing-6); /* 24px */
}

.button {
  padding: var(--spacing-3-5) var(--spacing-8); /* 14px 32px */
}
```

### Layout Gaps
```css
.section {
  display: flex;
  flex-direction: column;
  gap: var(--spacing-24); /* 96px */
}

.grid {
  display: grid;
  gap: var(--spacing-6); /* 24px */
}
```

### Margins
```css
.header {
  margin-bottom: var(--spacing-10); /* 40px */
}

.card {
  margin-left: var(--spacing-16); /* 64px */
}
```

### Section Spacing
```css
.hero {
  padding: var(--spacing-24) 0; /* 96px vertical */
  margin-bottom: var(--spacing-36); /* 144px */
}
```

## Responsive Spacing

Spacing can be adjusted for different breakpoints:

```css
.section {
  gap: var(--spacing-24); /* 96px on desktop */

  @media (--md-n-below) {
    gap: var(--spacing-12); /* 48px on mobile */
  }
}
```

## Best Practices

### Use Consistent Increments
Stick to the defined scale rather than arbitrary values. This ensures visual consistency across the application.

✅ Good:
```css
padding: var(--spacing-6); /* 24px */
```

❌ Bad:
```css
padding: 23px; /* Not in scale */
```

### Choose Appropriate Scale
- **0-16px**: Micro-adjustments, tight spacing, icons
- **20-48px**: Component padding, comfortable gaps
- **56-160px**: Section spacing, hero elements
- **176-384px**: Large layout spacing, major sections

### Combine for Complex Layouts
```css
.card {
  padding: var(--spacing-6) var(--spacing-8); /* 24px 32px */
  margin-bottom: var(--spacing-10); /* 40px */
  gap: var(--spacing-4); /* 16px */
}
```

### Responsive Adjustments
Always reduce spacing on smaller screens:

```css
.container {
  padding: var(--spacing-24);

  @media (--md-n-below) {
    padding: var(--spacing-12);
  }

  @media (--sm-n-below) {
    padding: var(--spacing-6);
  }
}
```

## Accessibility

Using rem-based spacing ensures:
- Scales with user font size preferences
- Maintains proportions when zooming
- Respects accessibility settings
- Consistent spacing across devices

All spacing values scale proportionally when users adjust their browser's default font size.
