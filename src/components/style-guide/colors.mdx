## Usage

All colors are available as CSS custom properties:

```css
.example {
  background: var(--bg-primary);
  color: var(--text-primary);
  border: 1px solid var(--border-default-color);
}

.button {
  background: var(--accent-primary);
  color: var(--text-primary);
}
```

## Accessibility

All color combinations meet WCAG 2.1 AA contrast requirements:

- **Text primary on bg-primary**: 19.5:1 (dark), 19.5:1 (light)
- **Accent primary on bg-primary**: 4.5:1+ (both themes)
- **Accent secondary on bg-primary**: 4.5:1+ (both themes)

Use the Accessibility panel in Storybook to verify color contrast for specific components.
